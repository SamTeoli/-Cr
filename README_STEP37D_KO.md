# 37D~37F — 적 턴 정렬·실행 파이프라인 묶음

## 37D: 적 행동 순서 계획

확정된 적 턴 규칙을 행동 계획 생성 단계에 적용합니다.

- 적 처리 순서: 왼쪽 → 중앙 → 오른쪽
- 각 적의 행동 순서: 이동 → 공격 → 능력(버프·디버프 포함)
- 같은 적의 같은 종류 행동은 외부 입력 순서를 유지
- 계획 생성 순간의 위치를 기준으로 순서를 고정
- 존재하지 않거나 쓰러졌거나 위치가 없는 적의 명령은 실행 전에 거부

## 37E: 정렬된 적 턴 실행 파이프라인

섞인 외부 명령을 순서 계획으로 만든 뒤 기존 적 턴 지휘 시스템으로 전달합니다.

- 계획 생성 실패와 실제 턴 실행 실패를 구분
- 계획, 표시용 의도, 실제 행동 결과의 순서가 같은지 추적
- 적 턴이 끝나면 기존 규칙대로 플레이어 턴을 시작

## 37F: 전체 회귀검증 편입

C01~C12 전체 런타임 회귀검증에 적 턴 정렬과 실행 파이프라인을 추가합니다.

적 행동의 종류·수치·대상은 이 묶음에서 만들지 않습니다. 외부 데이터가 만든 명령만
확정 규칙으로 정렬하고 기존 실행 계층에 연결합니다. 검증 코드의 ID와 수치는
연결 확인용이며 게임 데이터가 아닙니다.
