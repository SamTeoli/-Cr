# 23단계: E02 닳은 손잡이 공격 완료 후 반격

## 구현 범위

- 기존 enum 직렬화 순서를 보존하면서 `AttackCompleted` 사건을 추가한다.
- `AttackDeclared` 사건의 자식으로만 공격 완료 사건을 생성한다.
- 활성 E02가 장착된 공격 몬스터는 공격 완료 후 반격 1을 얻는다.
- 장착 몬스터별 플레이어 턴당 1회만 발동한다.
- 같은 완료 사건은 다른 턴에 다시 처리해도 중복 발동하지 않는다.
- 공격 선언만 있고 완료 사건이 없는 취소·무효·대상 상실 공격에는 발동하지 않는다.
- E02 비활성 상태에서는 완료 사건이 있어도 발동하지 않는다.
- 반격 적용 전후 수치를 `StatusApplied` 사건으로 기록한다.
- 공격 피해량과 반격 소비 규칙은 해당 실행기 단계에서 연결하며 여기서 임의로 정하지 않는다.
- C01~C12와 E01~E08 원본 데이터는 변경하지 않는다.

## 프로젝트에 적용

ZIP 안의 `Assets` 폴더를 Unity 프로젝트 최상위 폴더에 복사하고 파일 교체를 허용한다.

## Unity에서 확인

1. Unity 컴파일이 끝날 때까지 기다린다.
2. Console에 빨간 컴파일 오류가 없는지 확인한다.
3. `Have a Break > Validate E02 Worn Handle Attack Completion`을 누른다.
4. `E02 Worn Handle attack completion passed.`가 표시되는지 확인한다.
