# 6단계: 시작 패 교체

## 기준

통합 기획서 v0.4의 2.2 시작 패와 턴 드로우 규칙을 기준으로 한다.

## 구현 범위

- 시작 패 5장 중 0~5장을 선택할 수 있다.
- 시작 패 교체는 전투당 한 번만 확정할 수 있다.
- 선택한 카드는 `RedrawHolding` 영역에 임시로 분리한다.
- 남은 드로우 더미에서 선택 수만큼 새 카드를 먼저 받는다.
- 분리했던 카드를 드로우 더미로 되돌린 뒤 다시 섞는다.
- 새로 받은 카드는 다시 교체할 수 없다.
- 0장을 선택해 확정해도 교체 기회는 사용된다.
- 선택 중복, 시작 패가 아닌 카드, 남은 덱 부족은 상태를 변경하지 않고 실패한다.
- 같은 덱과 같은 시드는 같은 교체 결과와 덱 순서를 만든다.
- 교체 후에도 첫 플레이어 턴의 일반 드로우는 생략한다.

시작 패 교체는 일반 드로우가 아니며, 드로우 유발 효과와 사건 기록은 이후 효과·사건
시스템 단계에서 별도 종류로 연결한다.

## Unity에서 확인

1. 프로젝트를 연 뒤 스크립트 컴파일이 끝날 때까지 기다린다.
2. `Have a Break > Card Data Tools`를 연다.
3. `Validate Starting Hand Redraw`를 누른다.
4. `Starting hand redraw passed.`가 표시되는지 확인한다.
5. Console에 빨간 컴파일 오류가 없는지 확인한다.
