# 37A — 적 턴 지휘 시스템

## 구현 내용

- 외부에서 전달한 이동·공격·능력 명령을 순서대로 실행합니다.
- 이동은 기존 C08·C04·C12 흐름을 사용합니다.
- 공격은 기존 C09, 방어 소모, 초과 피해, 파괴 판정을 사용합니다.
- 능력은 기존 C10 취소 흐름을 사용합니다.
- 각 행동 뒤 쓰러진 적을 생존 목록과 위치에서 정리합니다.
- 플레이어 패배 또는 적 전멸이 확인되면 남은 명령을 실행하지 않습니다.
- 전투가 계속되면 적 턴을 완료하고 다음 플레이어 턴과 C11 효과를 해결합니다.

적 행동 선택이나 수치는 이 단계에서 만들지 않습니다. 적 데이터가 결정한 명령을
실행하는 지휘 계층만 추가합니다.

## Unity 검증

`Have a Break > Validate Runtime Enemy Turn Orchestration`

성공 메시지:

`Battle runtime enemy turn orchestration passed.`

검증은 이동→공격→능력 순차 실행과 C08·C09·C10 반응, 플레이어 턴 전환,
패배 시 남은 행동 중단을 확인합니다.
