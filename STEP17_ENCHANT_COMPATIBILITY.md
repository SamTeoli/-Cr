# 17단계: E01~E08 추가 호환 조건 판정

## 기준

E01~E08 시험 인첸트 제작서 v0.4의 `추가 호환 조건`과 C01~C12 카드 원문을
기준으로 한다.

## 구현 범위

- C01~C12에 카드 원문을 바꾸지 않는 구조화된 호환 태그를 저장한다.
- E01은 기본 생명력이 있는 몬스터만 허용한다.
- E02는 기본 공격력이 있는 몬스터만 허용한다.
- E03은 본 효과 완료 사건이 존재하는 카드만 허용한다.
- E04는 수록 기본 비용이 2 이상인 스킬만 허용한다.
- E05는 적에게 실제 영향을 주는 트랩만 허용한다.
- E06은 수치형 반복 효과가 있는 결계만 허용한다.
- E07은 정상 처리 후 묘지로 이동하는 스킬·트랩만 허용한다.
- E08은 적 1개를 선언할 때 고정하는 효과가 있는 카드만 허용한다.
- 카드 종류와 추가 호환 조건을 모두 통과해야 즉시 장착 대상으로 사용할 수 있다.
- E01~E08과 C01~C12의 전체 호환 행렬을 검증한다.
- 이름, 비용, 공격력, 생명력, 레벨과 효과 문구는 변경하지 않는다.
- C10 레벨 5처럼 전투 중 처리 후 영역이 달라지는 세부 조건은 실제 전투 레벨 상태를
  연결하는 후속 단계에서 다시 검사한다.

## Unity에서 진행

1. 프로젝트를 연 뒤 컴파일이 끝날 때까지 기다린다.
2. `Have a Break > Card Data Tools`를 연다.
3. `Apply Enchant Compatibility Tags C01-C12`를 한 번 누른다.
4. `Applied enchant compatibility tags to C01-C12.`를 확인한다.
5. `Validate Enchant Compatibility`를 누른다.
6. `Enchant compatibility passed.`가 표시되는지 확인한다.
7. Console에 빨간 컴파일 오류가 없는지 확인한다.
