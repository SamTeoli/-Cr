# 22단계: E08 노선 고정핀 위치 대상

## 구현 범위

- 적 필드의 왼쪽·중앙·오른쪽 위치 상태를 추가한다.
- 같은 적은 한 위치에만 존재하며 이미 점유된 위치에는 배치·이동할 수 없다.
- E08 활성 카드가 적 1개를 선언하면 그 적의 현재 위치를 기록한다.
- 효과 해결 시 선언한 적이 아니라 기록된 위치의 현재 점유자를 대상으로 반환한다.
- 해결 시 기록 위치가 비어 있으면 실패하며 다른 위치의 원래 적을 다시 찾지 않는다.
- 선언 당시 실제 적이 없는 위치는 선택할 수 없다.
- E08이 없으면 기존 방식대로 선언한 적 ID를 고정하며, 적이 이동해도 같은 적을 반환한다.
- C01~C12와 E01~E08 원본 데이터는 변경하지 않는다.

실제 이동·피해 카드 효과 실행기는 이후 카드별 효과 명령 단계에서 이 대상 해결기를
호출한다. 현재 단계에서는 기획에 없는 적 능력치나 행동을 추가하지 않는다.

## 프로젝트에 적용

ZIP 안의 `Assets` 폴더를 Unity 프로젝트 최상위 폴더에 복사한다. 이번 단계는 기존 파일을
교체하지 않고 새 위치·대상 구조와 Editor 검증기만 추가한다.

## Unity에서 확인

1. Unity 컴파일이 끝날 때까지 기다린다.
2. Console에 빨간 컴파일 오류가 없는지 확인한다.
3. `Have a Break > Validate E08 Route Pin Position Target`을 누른다.
4. `E08 Route Pin position target passed.`가 표시되는지 확인한다.
