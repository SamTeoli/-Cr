# 25단계: E05 녹슨 안내방송 영향받은 적 약화

## 구현 범위

- 적별 약화 수치를 관리하는 전투 상태와 레지스트리를 추가한다.
- 본 효과 완료 사건에 실제로 영향을 받은 적 ID 목록을 연결한다.
- 영향 목록은 생존 중인 적만 허용하며 빈 ID, 중복 ID와 존재하지 않는 적을 거부한다.
- 활성 E05 장착 트랩의 본 효과 완료 후 영향받은 현재 적 각각에게 약화 1을 적용한다.
- 영향받지 않은 적에게는 약화를 적용하지 않는다.
- 영향 목록이 비어 있거나 E05가 비활성 상태이면 발동하지 않는다.
- 같은 본 효과 완료 사건은 중복 처리하지 않는다.
- 약화 적용 전후 수치를 적별 `StatusApplied` 사건으로 기록한다.
- 트랩 자체의 대상 선택과 본 효과는 카드별 실행기 단계에서 연결하며 임의로 만들지 않는다.
- C01~C12와 E01~E08 원본 데이터는 변경하지 않는다.

## 프로젝트에 적용

ZIP 안의 `Assets` 폴더를 Unity 프로젝트 최상위 폴더에 복사한다. 이번 단계는 기존 파일을
교체하지 않고 새 영향 기록·적 상태·E05 해결기와 검증기만 추가한다.

## Unity에서 확인

1. Unity 컴파일이 끝날 때까지 기다린다.
2. Console에 빨간 컴파일 오류가 없는지 확인한다.
3. `Have a Break > Validate E05 Rusty Announcement Affected Enemies`를 누른다.
4. `E05 Rusty Announcement affected enemies passed.`가 표시되는지 확인한다.
