# 36D — 적 공격 피해·초과 피해·파괴 판정

## 구현 내용

- 배틀 런타임 상태에 기본 최대 체력 30의 플레이어 상태를 포함했습니다.
- 적 공격 스냅샷에 약화를 적용해 실제 공격력을 계산합니다.
- 몬스터 방어를 먼저 소모한 뒤 남은 수치만 생명력 피해로 적용합니다.
- 몬스터의 남은 생명력을 넘는 피해를 플레이어에게 전가합니다.
- 피해 사건 뒤 상태 기반 파괴 검사를 실행하고 `AttackCompleted` 사건을 기록합니다.
- 같은 공격 선언의 피해가 두 번 처리되지 않도록 차단합니다.

## Unity 검증

메뉴에서 다음 항목을 실행합니다.

`Have a Break > Validate Runtime Enemy Attack Damage`

성공 메시지:

`Battle runtime enemy attack damage passed.`

검증은 C09 레벨 5 방어 5의 소모, 몬스터 파괴를 넘는 피해 3의 플레이어 전가,
플레이어 기본 최대 체력 30, 약화 적용, 방어 우선 소모, 중복 해결 차단을 확인합니다.
