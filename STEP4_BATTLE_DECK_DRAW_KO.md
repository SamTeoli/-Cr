# 4단계: 전투 덱 생성과 드로우

## 구현 범위

- 전투 시작 시 지정한 시드로 드로우 더미 순서를 섞는다.
- 시작 패는 최대 5장이다.
- 플레이어의 첫 번째 턴 시작 드로우는 건너뛴다.
- 이후 플레이어 턴 시작에는 1장을 뽑는다.
- 패가 10장이면 드로우 더미의 맨 위 카드를 유지하고 `HandFull`을 반환한다.
- 드로우 더미가 비면 묘지를 섞어 새 드로우 더미로 만든 뒤 뽑는다.
- Unity 전역 난수 상태를 사용하지 않으므로 같은 덱과 같은 시드는 같은 순서를 만든다.

## Unity에서 확인

1. 프로젝트를 연 뒤 스크립트 컴파일이 끝날 때까지 기다린다.
2. `Have a Break > Card Data Tools`를 연다.
3. `Validate Battle Deck Draw`를 누른다.
4. `Battle deck draw passed.`가 표시되는지 확인한다.
5. Console에 빨간 컴파일 오류가 없는지 확인한다.

이 검증은 C01과 C05 시험 카드가 필요하다. 기존 C01~C12 에셋의 이름, 코스트, 수치,
효과 문구는 변경하지 않는다.
