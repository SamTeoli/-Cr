# 19단계: E04 예비 전원 전투 비용

## 구현 범위

- 전투카드ID와 런 카드 인첸트 상태를 연결하는 레지스트리를 추가한다.
- 활성 E04가 장착된 스킬의 현재 레벨 마력 비용을 1 낮춘다.
- E04 적용 직후 비용은 1 미만으로 내려가지 않는다.
- 미리보기와 최종 확정에서 같은 비용 계산기를 사용한다.
- 미리보기 후 E04 활성 상태가 달라지면 기존 미리보기는 무효가 된다.
- 확정된 감소 비용만 실제 마력에서 차감한다.
- E04 비활성 상태에서는 원래 비용을 사용한다.
- C01~C12와 E01~E08 원본 데이터는 변경하지 않는다.

## 프로젝트에 적용

ZIP 안의 `Assets` 폴더를 Unity 프로젝트 최상위 폴더에 복사하고 파일 교체를 허용한다.
기존 `BattleCardPlayState.cs`는 최신 19단계 파일로 교체된다.

## Unity에서 확인

1. Unity로 돌아가 스크립트 컴파일이 끝날 때까지 기다린다.
2. Console에 빨간 컴파일 오류가 없는지 확인한다.
3. 상단 메뉴 `Have a Break > Validate E04 Backup Power Battle Cost`를 누른다.
4. `E04 Backup Power battle cost passed.`가 표시되는지 확인한다.
