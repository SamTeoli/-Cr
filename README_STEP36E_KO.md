# 36E — 적 능력과 C10 자동 취소

## 구현 내용

- 적 턴 능력 선언·취소·완료 사건을 배틀 이벤트에 추가했습니다.
- 살아 있는 적이 사용하는 일반 공격 이외의 능력을 런타임에서 해결합니다.
- 설치된 C10을 최근 등록 순서부터 자동 탐색합니다.
- C10이 취소할 수 있는 능력이라면 능력 해결을 중단하고 카드 레벨 규칙을 적용합니다.
- C10 레벨 5의 패 복귀와 레벨 3 이상의 약화 1을 기존 확정 규칙 그대로 사용합니다.
- 광역 능력은 C10 레벨 4 이상에서만 취소됩니다.

## Unity 검증

`Have a Break > Validate Runtime Enemy Ability And C10`

성공 메시지:

`Battle runtime enemy ability and C10 passed.`

검증은 레벨 5 단일 대상 취소·패 복귀, 레벨 3 광역 능력 미취소,
레벨 4 광역 능력 취소·묘지 이동, 약화 1, 설치 등록부 정리를 확인합니다.
