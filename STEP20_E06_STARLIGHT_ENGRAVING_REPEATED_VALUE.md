# 20단계: E06 별빛 각인 반복 효과 수치

## 구현 범위

- 반복 효과 명령 생성 전에 사용할 구조화된 수치 묶음을 추가한다.
- 활성 E06이 장착된 호환 결계는 반복 효과의 첫 수치만 1 증가한다.
- 대상 수, 발동 횟수와 조건 기준값은 그대로 유지한다.
- E06이 비활성 상태이거나 전투카드 인첸트 상태가 등록되지 않았다면 원래 수치를 반환한다.
- C11과 C12 모두 같은 계산 경계를 사용한다.
- C11~C12의 이름, 비용, 레벨 수치와 효과 문구는 변경하지 않는다.
- 검증기의 숫자는 필드 보존을 확인하기 위한 독립 시험값이며 카드 데이터에 저장되지 않는다.

결계별 실제 반복 효과 명령 생성은 해당 카드 효과 실행기가 추가될 때 이 계산기를 호출한다.
현재 단계에서는 기획에 없는 대상 선택이나 발동 시점을 임의로 구현하지 않는다.

## 프로젝트에 적용

ZIP 안의 `Assets` 폴더를 Unity 프로젝트 최상위 폴더에 복사한다. 이번 단계는 기존 파일을
교체하지 않고 새 런타임 계산기와 Editor 검증 파일만 추가한다.

## Unity에서 확인

1. Unity로 돌아가 스크립트 컴파일이 끝날 때까지 기다린다.
2. Console에 빨간 컴파일 오류가 없는지 확인한다.
3. `Have a Break > Validate E06 Starlight Engraving Repeated Value`를 누른다.
4. `E06 Starlight Engraving repeated value passed.`가 표시되는지 확인한다.
