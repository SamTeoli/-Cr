# 21단계: E07 환승 도장 묘지 이동 대체

## 구현 범위

- 정상 처리 후 묘지로 이동하는 스킬·트랩에만 E07 대체 규칙을 적용한다.
- 전투당 처음인 경우 묘지 대신 드로우 더미 맨 아래로 이동한다.
- 대체 이동에 성공한 뒤에만 전투당 1회 표식을 소비한다.
- 두 번째 정상 묘지 이동부터는 원래대로 묘지로 이동한다.
- 파괴, 강제 이동 등 정상 처리 완료가 아닌 묘지 이동은 대체하지 않고 표식도 소비하지 않는다.
- 대체된 이동은 실제 목적지를 드로우 더미로 기록하며 묘지 이동 사건을 만들지 않는다.
- 임시 카드는 기존 규칙대로 묘지 대신 소멸하며 E07을 사용하지 않는다.
- C01~C12와 E01~E08 원본 데이터는 변경하지 않는다.

## 프로젝트에 적용

ZIP 안의 `Assets` 폴더를 Unity 프로젝트 최상위 폴더에 복사하고 파일 교체를 허용한다.

## Unity에서 확인

1. Unity 컴파일이 끝날 때까지 기다린다.
2. Console에 빨간 컴파일 오류가 없는지 확인한다.
3. `Have a Break > Validate E07 Transfer Stamp Graveyard Replacement`를 누른다.
4. `E07 Transfer Stamp graveyard replacement passed.`가 표시되는지 확인한다.
